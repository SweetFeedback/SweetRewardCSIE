h1. Gumball Machine Server-Side

By Louis Yu, Janet Huang, and many awesome contributors!

h2. Packages You need

You need to install an apache http-server on your local machine in order to run our server-side code. Two ways to install these packages:

1. Install the "XAMPP":http://www.apachefriends.org/en/xampp.html package here 
2. Install the *Apache Server* + *MySQL* + *PHP* seperately.


h2. Installation Steps.

1. Download the code from SweetRewardCSIE
* Download the whole *master.zip* file on the github.
* Pull the code from by using tool git 
@git pull https://github.com/louisccc/SweetRewardCSIE.git@

2. Move the directory *SweetRewardCSIE/* to apache webpage directory e.g. *www/* or *htdocs/*
3. Move the file *SweetRewardCSIE/php/config.php.template* to *config.php*
* Modify the strings of database information. 

4. Move the file *SweetRewardCSIE/collaboration/config.json.template* to *config.json*
* Modify the strings to the url of server.

5. Import the database schema to your local database.
* If you use command line, switch your workspace to *SweetRewardCSIE/* and then give command,
@mysql -u [username of your database] -p < schema.sql@
* If you use phpmyadmin to operate your local database, please select import button and choose the file *schema.sql* under *SweetRewardCSIE/*
